This repository contains code and documentation for a 3D medical image segmentation project using the Medical Segmentation Decathlon Task04 Hippocampus dataset.

The source code is licensed under the MIT License.

Some report figures may be derived from Medical Segmentation Decathlon images and labels. Those figures are included for research, educational, and portfolio documentation purposes. The original dataset should be credited when using or redistributing these materials.

No raw medical image dataset files, trained model checkpoints, or large generated prediction arrays are included in this repository.
